University Registration System
A Java-based University Registration System that allows students to view, register, and manage courses through a client-server architecture.
Technical Concepts Used
Object-Oriented Programming (OOP)
Encapsulation, Inheritance & Abstraction
Interfaces & Polymorphism
Socket Programming
Multi-threading
File Handling
Java Swing GUI
ArrayList Data Structure
Exception Handling
Main Features
Secure student login system
View available university courses
Register and drop courses
Display registered courses for each student
Course capacity management
Real-time client-server communication
Multi-client support using threads
File-based data storage using .txt files
Interactive GUI using Java Swing
Future Improvements
